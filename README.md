## Getting Started

### Prerequisites

- JDK 17 or higher installed.
- Maven (or use the included `mvnw` wrapper).

### Installation & Run

1. Clone the repository.
2. Navigate to the project root.
3. Run the application using the Maven Wrapper:

```bash
# Mac/Linux
./mvnw spring-boot:run

# Windows
mvnw spring-boot:run
```
